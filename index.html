Minha trajetória com a programação!

Para navegar entre os slides, você pode utilizar os botões ao lado ou as setas do seu teclado.

HTML e CSS

Comecei os estudos fazendo pequenos projetos estáticos para entender como funcionava cada coisa. Base muito importante que todo programador deve ter, pois com o HTML é construído o corpo do site e o CSS é o estilo.

JavaScript

Próximo passo para avançar foi o JavaScript para deixar de fazer sites estáticos. É uma linguagem que está presente tanto no Front End como no Back End e com ela pude evoluir para projetos um pouco mais ambiciosos, onde foquei bastante na maniuplação de elementos do DOM (Document Object Model).

Python

Depois de ter feito alguns projetos Front End, decidi praticar Python focando em automação de tarefas. Essa linguagem é muito poderosa e é possível fazer muita coisa com ela. Além dos projetos de automação trabalhei minha lógica de programação com alguns exercícios, assim como iniciei os estudos em Orientação a Objetos.

PHP

Trabalhei profissionalmente com essa linguagem, onde usei para se comunicar com um banco de dados para obter informações e exibir no Front End. Além da linguagem utilizei bastante o framework Laravel.
